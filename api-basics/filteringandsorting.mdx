---
title: 'Filtering and sorting'
description: 'Some of Mangopay’s endpoints support filtering and sorting features based on a given object’s parameters. This can prove useful to look for specific information and to improve performance.'
---


You can use query parameters to define the values by which to filter or sort information returned by the API.

**Filtering query string example:**

`…/sellers?Status=Onboarded,ToOnboard&Country=US`

**Sorting query string example:**

`…/sellers?Sort=CreationDate:ASC`
