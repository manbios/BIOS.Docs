## Environment

BIOS provide two environments:

<CardGroup cols={2}>
  <Card
    title="Production"
    icon="wave-pulse"
    href="/introduction#environment"
  >
  For live production
  ```javascript 
    api.enterbios.com
  ```
  </Card>
  <Card
    title="Sandbox"
    icon="screwdriver-wrench"
    href="/introduction#environment"
  >
  For testing and integration
  ```javascript 
    api.sandbox.enterbios.com
  ```
  </Card>
</CardGroup>

The BIOS API accepts and returns:<br/>
`Content-Type` application/json <br/>
`Encoding` UTF-8 JSON

<Note>	
**Note**<br/>
Requests to generate an OAuth token use the application/x-www-form-urlencoded Content-Type.<br/> See more information in OAuth 2.0 authentication.
</Note>
